# emerging_coronaviruses

To run the genbank_data_creation script please download the folder "Data" from here:
https://drive.google.com/drive/folders/1Ty-mPdDAKPGYe2M-SYzAJwEQByE433qq?usp=sharing. This folder has the raw data from Genbank, opened in Geneious Prime, and saved as.csv  

Place the folder in you project directory of preference and edit line 21 of of the script "genbank_data_creation.R" and you should be ready to go.
